# OrigamiObjectCreator
> ### 종이접기를 이용한 절차적 모델링
```
blender를 이용해서 쉽게 종이접기 모델링을 만들어 낼 수 있는 도구 입니다.
다양한 방법의 종이 접기 기능을 제공합니다.
```
